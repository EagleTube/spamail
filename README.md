# Get Started
pip -r reqirements.txt
<br>
Option(arguments)<br>
-u : URL<br>
-s : Sender<br>
-x : Template.html<br>
-r : Receiver (in TXT or email)<br>
--single : one time send mail<br>
--mass : mass sender multiple emails<br>
--singlemass : for spam single email followed by -c for count of spam<br>
-c : count<br>
